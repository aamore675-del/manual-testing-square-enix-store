# Manual Testing – Square Enix Store

## Overview
This project demonstrates manual QA testing on an e-commerce platform. It includes structured test cases, edge cases, and a sample bug report to showcase testing and defect tracking skills.

## Scope
The testing focuses on key user flows including:
- Product search functionality
- Account creation and login
- Shopping cart behavior
- Checkout process

## Contents
- Test cases covering positive and negative scenarios
- Edge case testing
- Sample bug report with steps to reproduce, expected vs actual results

## Tools & Skills Demonstrated
- Manual Testing
- Test Case Design
- Bug Reporting
- Analytical Thinking
- QA Documentation

## Goal
The goal of this project is to demonstrate a structured approach to testing, attention to detail, and the ability to identify and document defects effectively.
